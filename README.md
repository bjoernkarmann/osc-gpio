# OSC-GPIO

This project is for controlling gpio pins on a raspberyPi using osc messages.

## Run

```
$ npm start
```


## Getting started:

Download and install the osc-gpio

```
git clone https://github.com/bjoernkarmann/osc-gpio.git && cd osc-gpio && npm install
```

To start run

``npm start``



## OSC protocol

- port: 9998
- adress: /pin/state
